# Screenshots folder

After building the Power BI dashboard following `powerbi/build_guide.md`,
export 2-3 screenshots here (File -> Export -> PDF/Image, or just Win+Shift+S)
and reference them in the main README so the GitHub page shows the actual
dashboard, not just a description of it. Recommended:
- `dashboard_overview.png` — the full page with KPI cards + table
- `risk_table_detail.png` — close-up of the conditional-formatted watchlist table
